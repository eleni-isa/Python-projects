Data Dictionary
Input Curves — Editable
Units Held
The number of units of the ETF held in the portfolio on each date. This changes when the portfolio is rebalanced — i.e. when you buy or sell shares.
Price (USD)
The closing market price of the ETF in US dollars on each date, as reported by Yahoo Finance.
FX Rate
The GBP/USD exchange rate — how many US dollars one British pound buys. Used to convert the USD market value into GBP (the portfolio's base currency).
Derived Curves — Computed by the Dependency Graph
Market Value
The total value of the position in US dollars.
= Units × Price
MV Base CCY
Market Value converted into GBP (the base currency). This is the true portfolio value from a UK asset manager's perspective.
= Market Value ÷ FX Rate
Daily P&L
The change in GBP market value from the previous period. Positive means the position gained value; negative means it lost value. The first period is always zero.
= MV Base CCY(t) − MV Base CCY(t−1)
Scenario Comparison
Use Compare to overlay two scenarios side by side. The Diff columns show the difference (Right minus Left) for each derived metric. Green means the right scenario performed better; red means it performed worse. Input columns are hidden in compare mode to focus on the outcomes.
